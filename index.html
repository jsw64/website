Finished Projects

Here are some of my projects that I have made since starting college. The code can be found on my github page. Note that I have more exciting projects in the works, stay tuned!

To Do List App

A to do list app written in Python 3 using Tkinter GUI. Allows users to add items, delete items, cross off items from a list as well as save and load to do lists. Very handy for keeping track of daily tasks and events.

Simple ML Model

A simple machine learning model (sklearn, CountVectorizer, svm) that was trained to detect spam emails according to the email's subject line. Trained on 4500 emails that were spam or not spam, then tested on 1000 emails with 98% accuracy.

Hangman Game

Hangman is a game in which a player guesses letters of a word or lose a life. I recreated this game in C, allowing users to load in a dictionary of words and play hangman directly from the command line.
